# 📝 ToDoList Project
A Simple ToDo List to keep track of tasks.

# How to Use It

| Commands  | Result |
| ------------- | ------------- |
| new  | Add A Task  |
| list  | List All Tasks  |
| delete | Remove Specific Task |
| quit (or q) | Quit App |

---

`The app must be used with the browser's console opened.`
